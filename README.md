##Project Process – Excel Dashboard
1. Data Collection
Relevant business data such as sales, customers, products, regions, and dates was collected from reliable sources.
2. Data Cleaning
Missing values, duplicates, and errors were identified and removed.
Data was formatted correctly to ensure accuracy and consistency.
3. Data Preparation
The data was organized into structured tables.
Required calculated fields and columns were created for analysis.
4. KPI Definition
Key Performance Indicators (KPIs) were defined based on business requirements, including:
Total Sales
Total Profit
Profit Margin (%)
Total Orders
Top Product and Top Region
5. Pivot Table Creation
Pivot Tables were created to summarize and analyze data across different dimensions such as time, region, and product.
6. Dashboard Design
Charts, graphs, KPI cards, and visuals were designed with a clean and user-friendly layout.
7. Interactivity
Slicers and filters were added to allow users to dynamically explore the data.
8. Validation and Testing
Dashboard results were validated by cross-checking them with the raw data to ensure accuracy.
9. Final Reporting
The final dashboard was prepared in a management-ready format for effective decision making.
10. Documentation and Sharing
Project objectives, KPIs, and the complete process were documented, and the dashboard was shared on GitHub as a portfolio project.
